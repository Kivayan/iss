# ISS Terminal App

Animated terminal app that renders an ASCII ISS and live telemetry

![image](https://images2.imgbox.com/ab/14/Kmy8OhXJ_o.gif)

## Quick install

Install from GitHub Releases with one command.

### Linux

```bash
curl -fsSL https://raw.githubusercontent.com/kivayan/iss/main/scripts/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/kivayan/iss/main/scripts/install.ps1 | iex
```

run with `iss` in your terminal.
Quit with `q` or `ctrl+c`.
